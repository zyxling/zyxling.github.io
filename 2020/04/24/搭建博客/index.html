hexo初始化有问题的解决方案：
依次输入:

  • npm audit fix
  • npm audit fix –force
  • npm audit之后再输入hexo init
    上博客之前在myHexo文件夹中，打开终端
  1. hexo clean:清理缓存
  2. hexo generate：生成hexo
  3. hexo deploy:部署hexo
