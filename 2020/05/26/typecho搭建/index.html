typecho搭建

typecho搭建

1.Apache的安装与使用

下载地址：https://www.apachelounge.com/download/

1.打开conf/heepd.conf修改Apache的配置

Define SRVROOT “C:/Apache24”

ServerRoot “${SRVROOT}”

2.在文件末尾添加：

注：#意思是注释

# php7 support C:\Apache24\php7

LoadModule php7_module “C:/Apache24/php7/php7apache2_4.dll”

AddType application/x-httpd-php .php .html .htm

#configure the path to php.ini

PHPIniDir “C:/Apache24/php7”

3.安装成功后，运行Apache24/bin/httpd.exe,在浏览器地址栏输入localhost，如果页面出现It works则Apache运行成功

2.PHP的安装与使用

1.PHP下载时最好从官网下载，否则可能出现找不到php7apache2_4.dll扩展的情况

下载地址：https://windows.php.net/download

2.复制将 PHP 的根目录下的 php.ini-development复制一份并改名为 php.ini，作为PHP的配置文件；

3.打开php.ini，修改配置信息

把; extension_dir = “ext” 改为extension_dir = “C:\Apache24\php7\ext”

4.修改需要加载的扩展文件

注：.ini文件的注释是分号（英文分号），所以取消注释就删除分号；

;extension=bz2

;extension=curl

;extension=ffi

;extension=ftp

extension=fileinfo

extension=gd2

;extension=gettext

;extension=gmp

;extension=intl

extension=imap

;extension=ldap

;extension=mbstring

;extension=exif ; Must be after mbstring as it depends on it

extension=mysqli

extension=oci8_12c ; Use with Oracle Database 12c Instant Client

;extension=odbc

extension=openssl

;extension=pdo_firebird

extension=pdo_mysql

;extension=pdo_oci

;extension=pdo_odbc

;extension=pdo_pgsql

;extension=pdo_sqlite

;extension=pgsql

;extension=shmop

; The MIBS data available in the PHP distribution must be installed.

; See http://www.php.net/manual/en/snmp.installation.php

;extension=snmp

;extension=soap

extension=sockets

;extension=sodium

;extension=sqlite3

;extension=tidy

;extension=xmlrpc

;extension=xsl

5.在Apache/htdocs里新建index.php，写入

6.重启服务器，地址栏输入localhost/index.php，如果出现

则代表运行成功

3.部署typecho

1.删除index.php,把build文件里的所有内容拖入htdocs里，注意不要拖入build文件夹，省的还得重配路径

2.重启服务器，打开浏览器，在地址栏输入127.0.0.1就可以看到初始的typecho了
